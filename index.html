Калькулятор прокачки для игры Мир танков
Калькулятор прокачки предназначен для расчёта количества опыта, необходимого для исследования того или иного танка. Если просто навести курсор на изображение танка, то подсветится оранжевым путь до самого младшего танка в ветке, а во всплывающем окне отобразится информация о количестве опыта, нужном для его исследования - минимальном и максимальном. Количество опыта, которое нужно для исследования лишь необходимых модулей и танков по пути к нужному, будем считать минимальным. Количество опыта, необходимое для исследования всех модулей и танков по пути к нужному, будем считать максимальным.

Если вам нужно посчитать количество опыта для исследования не от первого танка ветки, надо кликнуть по нужному танку, а потом по тому, от которого необходимо посчитать количество опыта. Или можно сделать наоборот - сначала кликнуть по танку, от которого нужно начать исследование, а потом кликнуть по танку, до которого необходимо посчитать количество опыта.

При клике по изображению танка его задний фон закрашивается оранжевым цветом. Как только вы выберете второй танк, то во всплывающем окне будет показано минимальное количество опыта и названия всех танков, которые придётся исследовать по пути к нужному. Чтобы увидеть максимальное количество опыта, нужно кликнуть по этому окну. После этого вы можете либо закрыть окно, либо просто кликнуть по любому танку, и окно закроется само.

Если вы выбрали танки, которые находятся в разных ветках, то будет показано сообщение об ошибке. Его можно либо закрыть, либо кликнуть по любому танку, и окно закроется само.

В верхней части экрана располагается меню для выбора наций, каждая из которых отображена своим флагом, кликнув по которому, вы попадёте на страницу с ветками исследования танков.

В данный момент сайт работает в тестовом режиме - настраивается работа скриптов, добавляется контент и т.д. Данные, выдаваемые во всплывающих окнах, могут быть не точны.
